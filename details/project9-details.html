Brain Tumor Detection System

The Brain Tumor Detection System is an AI-powered application designed to classify brain tumors using deep learning. It leverages a pre-trained TensorFlow/Keras model to analyze medical images and accurately identify tumor types.

Project Overview:

This desktop application, developed with Python and Tkinter, allows users to upload an image of a brain scan. The system processes the image and predicts whether a tumor is present, along with its classification into one of the following categories:

  • Glioma
  • Meningioma
  • No Tumor
  • Pituitary
