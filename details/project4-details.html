Project Description

A smart AI-based object detection system built using YOLOv3 and Python. This desktop application detects and classifies multiple objects in images or real-time video streams with high accuracy. Features include real-time detection, support for custom models, bounding boxes with labels, and an intuitive user interface.

Key Features

Real-Time Detection

Detects multiple objects in real-time using webcam or video input with high accuracy.

Custom Models

Supports switching between pre-trained models or using custom trained models.

Bounding Boxes

Displays object names and confidence scores directly on the image/video.

Intuitive UI

Clean user interface with dark theme support for better user experience.

Cross-platform

Works smoothly on both Windows and Linux operating systems.

High Performance

Optimized for efficient object detection with minimal resource usage.
